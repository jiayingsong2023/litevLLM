# Multi-modal Inference

FastInference (vLLM Lite) provides a streamlined framework for multi-modal inference, enabling models like Qwen2-VL to process images and text with high efficiency.

## 🚀 Real-Image Processing Pipeline

Unlike basic implementations that require pre-tokenized vision data, FastInference supports direct media input through its **MultiModalInputProcessor**.

### 1. Direct PIL Image Support
You can pass raw `PIL.Image` objects directly to the engine.
- **Auto-Preprocessing**: The `ImageProcessor` handles resizing, normalization, and color space conversion.
- **GPU Acceleration**: Tensors are moved to GPU memory immediately after preprocessing.

### 2. High-Throughput Vision Attention
Our **Triton PagedAttention** kernel is optimized for the long-sequence context typically generated by vision encoders (e.g., 576 tokens per image).
- **Benchmarked Speed**: **532 TPS** at Batch Size 32 with 576 vision tokens per request.

## 🛠 Usage Example

```python
from PIL import Image
from vllm import LLM

# Initialize multi-modal model
llm = LLM(model="Qwen/Qwen2-VL-7B-Instruct")

# Load a real image
image = Image.open("images/sample.jpg")

# Inference with raw image object
prompt = "Describe the contents of this image."
outputs = llm.generate(prompt, multi_modal_data={"image": image})

for output in outputs:
    print(output.outputs[0].text)
```
